# Dispaying Messages to Users
## Write() , WriteLine()
```CSharp
Console.WriteLine("Hello World: {0}h, 1000); 
```
## Escape Sequences
- \t : print a Tab
- \n : print in newline
- \\\ : To print the backslash character itself (\\) 
```CSharp
Console.WriteLine("I am 5’9\" tall");
// Output: Console.WriteLine(“I am 5’9\” tall”);
```
